# Restaurant Management System

A full-stack application designed to manage restaurant operations, including orders, reservations, and table management.

## Features
* **Order Management**: Track and update the status of incoming orders.
* **Reservation System**: Handle table bookings and customer reservations.
* **Table Management**: Organize restaurant seating arrangements.
* **Menu Control**: Manage menu items and pricing.

## Tech Stack
* **Frontend**: React.js
* **Backend**: Node.js, Express
* **Database**: MongoDB

## Setup Instructions
1. Clone this repository: `git clone https://github.com/karthikeya871/restaurant-system.git`
2. Install dependencies for the server: `npm install`
3. Install dependencies for the client: `cd admin-panel && npm install`
4. Start the backend: `node app.js`
5. Start the frontend: `npm start` (inside the `admin-panel` directory)
